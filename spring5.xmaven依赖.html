Spring 5.x Maven依赖管理器

查找和使用Spring 5.x的核心Maven依赖项

Spring Core

Spring框架的核心工具，包含控制反转(IoC)和依赖注入(DI)

Maven依赖: 
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-core</artifactId>
    <version>5.3.23</version>
</dependency>

Spring Context

基于Core构建的应用程序上下文，提供企业级服务

Maven依赖: 
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-context</artifactId>
    <version>5.3.23</version>
</dependency>

Spring Web MVC

用于构建Web应用程序的MVC框架

Maven依赖: 
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-webmvc</artifactId>
    <version>5.3.23</version>
</dependency>

Spring Data JPA

简化基于JPA的数据访问层的实现

Maven依赖: 
<dependency>
    <groupId>org.springframework.data</groupId>
    <artifactId>spring-data-jpa</artifactId>
    <version>2.7.5</version>
</dependency>

Spring Security

提供身份验证和授权功能的安全框架

Maven依赖: 
<dependency>
    <groupId>org.springframework.security</groupId>
    <artifactId>spring-security-web</artifactId>
    <version>5.7.5</version>
</dependency>

Spring Boot Starter

Spring Boot的快速启动依赖，包含自动配置

Maven依赖: 
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
    <version>2.7.5</version>
</dependency>

Maven配置示例

完整的pom.xml配置示例：

<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 
         http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>
    
    <groupId>com.example</groupId>
    <artifactId>spring-demo</artifactId>
    <version>1.0.0</version>
    <packaging>jar</packaging>
    
    <properties>
        <spring.version>5.3.23</spring.version>
        <spring.security.version>5.7.5</spring.security.version>
        <spring.data.version>2.7.5</spring.data.version>
    </properties>
    
    <dependencies>
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-core</artifactId>
            <version>${spring.version}</version>
        </dependency>
        
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-context</artifactId>
            <version>${spring.version}</version>
        </dependency>
        
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-webmvc</artifactId>
            <version>${spring.version}</version>
        </dependency>
        
        <dependency>
            <groupId>org.springframework.data</groupId>
            <artifactId>spring-data-jpa</artifactId>
            <version>${spring.data.version}</version>
        </dependency>
        
        <dependency>
            <groupId>org.springframework.security</groupId>
            <artifactId>spring-security-web</artifactId>
            <version>${spring.security.version}</version>
        </dependency>
    </dependencies>
</project>

使用说明

  • 确保使用与Spring 5.x兼容的依赖版本
  • Spring Boot可以简化版本管理，通过starter parent自动管理版本
  • 建议使用Maven的properties部分定义版本号，便于统一管理
  • 某些Spring项目可能需要额外的依赖（如Spring Security Config）
